# Інформаційна безпека телекомунікаційних та хмарних технологій

**Курс для магістрів спеціальності "Кібербезпека"**  
**AWS Academy Integration**

## Опис курсу

Курс охоплює комплексний підхід до забезпечення інформаційної безпеки в телекомунікаційних системах та хмарних середовищах, з акцентом на практичні навички роботи з AWS.

## Структура курсу

- **11 лекцій**
- **2 модульні контролі**
- **Інтеграція з AWS Academy**

## Модуль 1: Основи безпеки телекомунікацій та AWS
1. Вступ до ІБ телекомунікацій та хмар
2. Архітектура телекомунікаційних систем і вектори атак
3. Мережева безпека в AWS
4. Управління ідентифікацією та доступом (IAM)
5. Захист даних і криптографія в AWS

## Модуль 2: Захист сервісів, інциденти та відповідність
6. Безпека обчислювальних сервісів і контейнерів
7. Безпека веб- та телеком-застосунків
8. Моніторинг, логування та виявлення загроз
9. Захист від DDoS та забезпечення доступності
10. Реагування на інциденти та цифрова криміналістика
11. Відповідність стандартам і сучасні підходи
